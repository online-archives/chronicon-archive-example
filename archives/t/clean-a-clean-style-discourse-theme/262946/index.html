Clean is a clean style discourse theme. It doesn’t make massive changes to discourse’s default theme, so it should be compatible with most widgets.

The design concept of Clean comes from Ant-Design. When designing the theme, I almost deleted all the parts with gray, so that the whole theme looks bright and clean.
