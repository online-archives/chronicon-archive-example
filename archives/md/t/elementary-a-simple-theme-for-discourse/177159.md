# Elementary, a simple theme for Discourse

> **Category:** Theme
> **Author:** meghna
> **Created:** 2021-01-25 10:11

---

### Post #1 by [meghna](../../users/meghna.md)
*Posted: 2021-01-25 10:11*

Inspired from [Sam’s simple theme](https://meta.discourse.org/t/sams-personal-minimal-topic-list-design/23552) I wanted to create a theme that looked simple and at the same time had modern appeal. The end result was “elementary” theme.

This two column theme tries to cover all the basic information of a topic under topic title, and the “Last Post” column contains last poster details and last posted date.
