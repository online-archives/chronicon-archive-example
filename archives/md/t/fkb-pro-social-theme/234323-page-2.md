> Any way to hide the image preview / make it smaller?

You can change the image height 🔽

You need to create a new component for this. ![:slightly_smiling_face:](../../../assets/emoji/slightly_smiling_face.png)

  1. Go to `/admin/customize/themes/`  
Customize → Themes

  2. Click the **Components** tab and then the `Install` button

  3. On the popup window click `Create new` button and type the new component name.  


[![Screenshot 2023-02-15 at 19.03.01](../../../assets/images/234323/15e5b554e2dee0d995549b7ab45624c314c94803_2_517x246.png)](../../../assets/images/234323/15e5b554e2dee0d995549b7ab45624c314c94803.png "Screenshot 2023-02-15 at 19.03.01")

  4. Click `Create` button.

  5. The component created. Now select FKB Pro theme to activate it.  
![Screenshot 2023-02-15 at 19.04.28](../../../assets/images/234323/55706a597a0630bcf7d002cdf9805a8eccff83f0.png)

  6. Click the `Edit CSS/HTML` button.  
![Screenshot 2023-02-15 at 19.05.17](../../../assets/images/234323/4c94aa26d39f87ae43d939bdca2a1e1ca0eb3647.png)

  7. Click the **Common** tab and paste the below code to the **CSS** section.  


[![Screenshot 2023-02-15 at 19.09.37](../../../assets/images/234323/1386e838b0b63056c270a630d88d6128051a6922_2_517x236.png)](../../../assets/images/234323/1386e838b0b63056c270a630d88d6128051a6922.png "Screenshot 2023-02-15 at 19.09.37")




If you want to set the image height separately on desktop and mobile, use those tabs instead of Common. Common means this code will be active on desktop and mobile too.
    
    
    .topic-list {
      .main-link {
        .link-middle-line {
          .topic-image {
            height: 280px; // Default 300px
          }
        }
      }
    }
    

If you want to hide the image, use this 🔽
    
    
    .topic-list {
      .main-link {
        .link-middle-line {
          .topic-image {
            display: none;
          }
        }
      }
    }
